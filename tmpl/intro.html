“Oh My RSS - 这才是我要的RSS”

预计阅读时间 0 分钟（共 0 个字， 0 张图片， 0 个链接）

Oh My RSS 是一个 开源 的在线RSS服务，通过爬虫汇聚互联网上的精华内容，旨在为国内的 IT 从业者提供一个优质的学习圈子。

主要功能特性如下：

一、持续扩充的订阅源

为了确保内容质量，Oh My RSS 对订阅源有严格的筛选标准：
  • （1）巨头公司的技术博客
  • （2）技术大拿的博客
  • （3）互联网名站的热门
一些支持的订阅源：

CoolShell 、 科技爱好者周刊 、 Github热门 、 博客园精华 、 编码美丽 、 ...

二、VimLike的快捷键

Vim 党的福音来了~
n 下一篇文章（next） N 上一篇文章 F 切换全屏（fullscreen）
p 下一页（page） P 上一页 D 标记本页已读并翻页
j 向下移动一行 k 向上移动一行 space 向下翻一页
gg 回顶部 G 去底部 f 链接全览

三、Webapp助力隐私保护

基于 Web 本地存储技术构建，无需登录注册，从源头杜绝隐私问题。
